# README for the `tech-docs` Directory

This directory organizes the technical documentation under development for Project Tapestry, including requirements analysis, architecture design, decision records, governance, and work group documentation.

| Directory | Description |
| :-------- | :---------- |
| [`architecture`](architecture) | TVA methodology, phase outputs, Architectural Thesis, ADRs, diagrams |
| [`governance`](governance) | Anti-capture principle, participation criteria, federation model |
| [`work-groups`](work-groups) | Each work group's documentation |
| [`strategic-plan`](strategic-plan) | Overall strategy for execution |
| [`tapestry-reference`](tapestry-reference) | Documents for deploying and using Tapestry software |
