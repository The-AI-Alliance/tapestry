# README for the Evaluation Requirements Work Group

(This work might be done as part of the other work groups.)

Identify the requirements for all aspects of evaluation.

## Key Questions

1. What domain-specific, culturally-situated benchmarks are needed to evaluate performance in healthcare, education, agriculture, governance, and climate?
2. How are "culturally appropriate outputs" operationalized as a measurable metric, and who governs the assessment?
3. How is sovereignty compliance evaluated — what constitutes a passing data residency audit? A successful exit drill?
4. What automated evaluation can run continuously vs. what requires Cultural Review Board assessment?
5. How are the ≥15% domain performance and ≥20% cultural task targets measured and reported?
6. How is constitutional AI configuration audited for completeness and correctness?
7. What hallucination rate evaluation standards apply to Transformer vs. JEPA architecture nodes?
