# README for the Data Engineering Work Group

The team implementing the data infrastructure.

They will begin iterating the construction of the data infrastructure, based on requirements that are still forming:
* Identify candidate tools and techniques.
* Prototype and iterate on pipelines and management infrastructure towards meeting anticipated requirements.
* Identify and plan for delivery milestones.

## Preliminary Scope

- Sovereign data ingestion pipelines with residency enforcement
- Consent tracking and revocation infrastructure
- Sacred knowledge boundary enforcement at the pipeline level
- Bias detection and cultural representation auditing tooling
- Data export for the 72-hour exit procedure
