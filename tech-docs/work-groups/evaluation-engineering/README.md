# README for the Evaluation Engineering Work Group

The team implementing the evaluation capabilities.

Implement the evaluation requirements:
* Identify candidate tools and techniques.
* Prototype and iterate on the evaluation stack towards meeting anticipated requirements.
* Identify and plan for delivery milestones.

## Preliminary Scope

- Domain-specific, culturally-situated benchmark suites for the five target domains
- Automated sovereignty compliance verification (data residency, exit capability, island mode)
- Cultural appropriateness scoring pipeline with Cultural Review Board interfaces
- Continuous bias and representation auditing
- Constitutional AI configuration validation tooling

## Source Code

Work product spans all three subsystems; primary tooling in [`src/tapestry/`](../../../src/tapestry/).

**Status:** Early formation. Begins active implementation once Evaluation Requirements are ratified.
