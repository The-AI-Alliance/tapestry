# README for the Infrastructure Requirements Work Group

Identify the requirements for infrastructure to run and manage Tapestry.

## Key Questions

1. What are the minimum hardware specifications for a sovereign node capable of meeting the 72-hour exit guarantee?
2. What network isolation mechanisms are required for island mode operation?
3. What hardware-level data residency enforcement is required (vs. software-only enforcement)?
4. What are the deployment requirements for a 3-command sovereign node setup from the reference implementation?
5. What monitoring, alerting, and audit logging infrastructure must be available at each sovereign node?
6. What infrastructure is required to support both air-gapped enterprise deployments and networked federated training?
7. What are the infrastructure implications of supporting heterogeneous hardware (on-premise enterprise vs. national data center vs. cloud-sovereign)?
